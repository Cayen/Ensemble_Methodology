# Ensemble Methodology
**Multi-Model AI Economic Convergence Research**
**Last updated: July 4, 2026 | Wave 8 Complete**

---

## What This Project Is

An 8-model AI ensemble tracking converging systemic stressors 
in the US and global economy. Eight AI models — Claude, ChatGPT, 
Gemini, Grok, Copilot, Kimi, DeepSeek, and a data analysis AI — 
run in parallel on a shared research question, with structured 
protocols designed to prevent the failure modes that make 
multi-model research unreliable.

The core thesis: multiple simultaneous systemic stressors 
(private credit, agricultural, geopolitical, labor, AI/tech, 
currency) are interacting in ways that warrant rigorous 
independent tracking.

The core methodology: verify everything. Agree on nothing 
until a primary source says so.

---

## What This Project Learned About Itself

Early waves produced confident-looking output with real problems:
- Numbers that drifted between models with no one checking
- Ranges collapsed into fake single points
- The ensemble's own framework constructs presented as 
  market facts
- Eight models agreeing on an inherited wrong number, 
  mistaken for independent confirmation
- Important threads going quiet and disappearing without 
  anyone noticing

These failures produced the methodology documents now in 
this repo. The project stress-tested itself, found the 
failure modes, and built structural fixes.

The 72-78% convergence probability from early waves has 
been formally withdrawn. It had no methodology behind it. 
Current assessment: "Active multi-system stress with 
uncertain severity and timing."

## How to Use This Methodology

The gate and adversarial protocol are designed to be 
model-agnostic. You don't need eight AI models to apply them. 
You need:

1. A verification gate that tags every claim before it 
   moves forward
2. A forced-disagreement mechanism so silence can't 
   substitute for agreement
3. A topic registry that tracks what matters independent 
   of what's recent
4. Someone willing to go find the primary source

The fourth one is the hardest. It's also the only one 
that can't be automated.

---

## On Probability Framing

This project does not assign percentage probabilities to 
convergence scenarios. The 72-78% figure from early waves 
was a "vibe with a percent sign on it" — no base rate, 
no backtest, no methodology. It has been withdrawn.

What the evidence supports: multiple real stressors exist, 
interact, and reinforce each other. Whether they produce 
a discrete systemic break in Q3-Q4 2026 remains unproven.

Honest uncertainty is more useful than false precision.
