## **I. METHODOLOGY FRAMEWORK (V4)**

### **4-Tier Gate System**

| Tier | Definition | Burden of Proof | Use Case |
|------|-----------|-----------------|----------|
| **`[SOURCED]`** | Single attributable claim | Minimal inference | "Reuters reported X" |
| **`[VERIFIED]`** | Multiple independent sources agree | Cross-source confirmation | CNN + CBS + BBC confirm X |
| **`[INTEGRATED]`** | Cross-vector significance established | Multi-source + mechanism clarity | X affects Vectors 3, 5, 9 |
| **`[OPERATIONAL]`** | Safe for ensemble model assumptions | `[INTEGRATED]` + temporal stability | "Assume X holds unless contradicted" |

| Tag | Definition |
|-----|-----------|
| `[SOURCED-PENDING]` | Claimed but awaiting explicit source |
| `[SUPPORTED ANALYSIS]` | Reasonable inference from sourced claims |

| `[ENSEMBLE-DEFINED]` | Framework-internal analytical construct |

| `[UNVERIFIED]` | No source check attempted |
